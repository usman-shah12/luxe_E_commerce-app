# LUXE Shop

A simple single vendor e-commerce app built with Flutter using GetX architecture.  
This project is created for portfolio purposes.

## Features

- Splash Screen
- Onboarding Screen
- Login / Signup UI
- Home Screen with banner and product listing
- Product Details Screen
- Add to Cart
- Cart Screen with quantity update
- Checkout Flow
- Order Success Screen
- Wishlist
- Search
- Profile Screen

## Architecture

The project follows GetX architecture with:

- State Management: GetX
- Routing: GetX
- Dependency Injection: GetX
- Clean folder structure (modules, data, widgets)
